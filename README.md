# Mymemo
ブックマーク管理とメモ帳が同時に使える

# 簡単な説明

よく使うWebページを保存したり、また見たいWebページを保存出来る。
また、メモ帳として記録を残すことが出来る。

# サイトURL
https://mymemor.herokuapp.com/

# 機能一覧

- インクリメンタルサーチ）
- いいね機能
- 投稿機能
- ログイン機能
- フラッシュメッセージ

# 使用技術
- mysql2 (0.5.3)
- rails (5.2.4.2)
- ruby (2.5.1)
- デプロイ heroku
